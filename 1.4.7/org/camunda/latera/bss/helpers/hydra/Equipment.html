          Output execution variables:
          • homsOrderData*Equipment**Id BigInteger
          • homsOrderData*Equipment**Name String
          • homsOrderData*Equipment**Code String
          • homsOrderData*Equipment**Serial String

          prefix - String. IP address prefix. Optional. Default: empty string
          equipmentPrefix - String. Equipment prefix. Optional. Default: empty string
          equipmentSuffix - String. Equipment suffix. Optional. Default: empty string
          bindAddrType - String. Object-address bind type without 'BIND_ADDR_TYPE_' part. Optional. Default: 'Serv'
          addrType - String. Address type without 'ADDR_TYPE_' part. Optional. Default: '' (street address)
          beginDate - Any date type. Object address begin date. Optional. Default: null
          endDate - Any date type. Object address end date. Optional. Default: null
          operationDate - Any date type. Date which should overlap between object address begin and end dates. Optional. Default: current datetime
          isMain - Boolean. If true search only main object addresses, if false search only non-main object addresses subscriptions, if null - disable filter. Optional. Default: true

        • String calcAddress(Map input = [:])

          Calculate full entity address from its parts from execution variables

          For internal usage only

          Input execution variables:

          if addrType == '' (street address):

          • homsOrderData*%Entity%*%BindAddrType%RegionId BigInteger. Address region id
          • homsOrderData*%Entity%*%BindAddrType%State String. Parent region 'code' field with level 'state'
          • homsOrderData*%Entity%*%BindAddrType%StateType String. Parent region type code with level 'state'
          • homsOrderData*%Entity%*%BindAddrType%Oblast String. Parent region 'code' field with level 'oblast'
          • homsOrderData*%Entity%*%BindAddrType%OblastType String. Parent region type code with level 'oblast'
          • homsOrderData*%Entity%*%BindAddrType%District String. Parent region 'code' field with level 'district'
          • homsOrderData*%Entity%*%BindAddrType%DistrictType String. Parent region type code with level 'district'
          • homsOrderData*%Entity%*%BindAddrType%City String. Parent region 'code' field with level 'city'
          • homsOrderData*%Entity%*%BindAddrType%CityType String. Parent region type code with level 'city'
          • homsOrderData*%Entity%*%BindAddrType%Building String. Parent region 'code' field with level 'building'
          • homsOrderData*%Entity%*%BindAddrType%BuildingType String. Parent region type code with level 'building'
          • homsOrderData*%Entity%*%BindAddrType%Home String. Parent region 'home' with level 'building'
          • homsOrderData*%Entity%*%BindAddrType%Corpus String. Parent region 'corpus' with level 'building'
          • homsOrderData*%Entity%*%BindAddrType%Construct String. Parent region 'construct' with level 'building'
          • homsOrderData*%Entity%*%BindAddrType%Ownership String. Parent region 'ownership' with level 'building'
          • other region hierarchy levels - see Region.getRegionHierarchy
          • homsOrderData*%Entity%*%BindAddrType%Entrance String. Entrance number
          • homsOrderData*%Entity%*%BindAddrType%Floor Integer. Floor number
          • homsOrderData*%Entity%*%BindAddrType%Flat String. Flat number

          if addrType != '':

          • homsOrderData*%Entity%*%AddrType% String. Address code

          Output execution variables:

          if addrType == '' (street address):

          • homsOrderData*%Entity%*%BindAddrType%Address String. Same as result value

          Parameters:
          prefix - String. Address prefix. Optional. Default: empty string
          entityPrefix - String. Entity prefix. Optional. Default: empty string
          bindAddrType - String. Entity-address bind type without 'BIND_ADDR_TYPE_' part. Optional. Default: 'Actual'
          addrType - String. Address type without 'ADDR_TYPE_' part. Optional. Default: empty string (street address)
          Returns:
          String with full address, e.g. 'Russia, Moscow city, Zavodskaya st., bldg. 1, corp. 2, flat 5' if addrType == '' (street address), otherwise '127.0.0.1'
          See Also:
          Address.calcAddress

        • Boolean closeAddress(Map input = [:])

          Close entity address and fill up execution variables

          For internal usage only

          Input execution variables:

          if addrType == '' (street address):

          • homsOrderData*%Entity%Id BigInteger. Entity id
          • homsOrderData*%Entity%Type BigInteger. Entity type (ref code). Optional
          • homsOrderData*%Entity%*%BindAddrType%AddressId BigInteger. Entity address id

          if addrType != '':

          • homsOrderData*%Entity%Id BigInteger. Entity id
          • homsOrderData*%Entity%Type BigInteger. Entity type (ref code). Optional
          • homsOrderData*%Entity%*%AddrType% String. Address code. Optional if '%AddrType%Id' is set
          • homsOrderData*%Entity%*%AddrType%Id BigInteger. Entity address id. Optional if '%AddrType%' is set

          Output execution variables:

          if addrType == '' (street address):

          • homsOrderData*%Entity%*%BindAddrType%AddressClosed Boolean. Same as result value

          if addrType != '':

          • homsOrderData*%Entity%*%AddrType%Closed Boolean. Same as result value
          Parameters:
          prefix - String. Address prefix. Optional. Default: empty string
          entityPrefix - String. Entity prefix. Optional. Default: empty string
          bindAddrType - String. Entity-address bind type without 'BIND_ADDR_TYPE_' part. Optional. Default: 'Actual'
          addrType - String. Address type without 'ADDR_TYPE_' part. Optional. Default: empty string (street address)
          endDate - Any date type. Entity address end date. Optional. Default: current datetime
          isMain - Boolean. If true search only main entity addresses, if false search only non-main entity addresses if null - disable filter. Optional. Default: true
          Returns:
          True if entity address was closed successfully, false otherwise
